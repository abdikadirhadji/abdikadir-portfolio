# Project Screenshots

## How to Add Project Screenshots

1. **Take screenshots** of your projects:
   - Property Management System
   - SARMS System

2. **Save them** in this `projects` folder with these exact names:
   - `property-management.jpg` (or `.png`)
   - `sarms-system.jpg` (or `.png`)

3. **Recommended specifications:**
   - Format: JPG or PNG
   - Size: 1200x800px or larger
   - Aspect ratio: 3:2 or 16:9
   - Quality: High resolution for best display

4. **The portfolio will automatically:**
   - Display your screenshots in project cards
   - Fall back to icons if images aren't found
   - Optimize for web display

## File Structure

```
public/
  └── projects/
      ├── property-management.jpg  ← Your screenshot here
      └── sarms-system.jpg         ← Your screenshot here
```

## Tips

- Use clean, professional screenshots
- Show the main interface/dashboard
- Avoid personal information in screenshots
- Use consistent styling across screenshots
